（船新版本）基于Hexo的matery主题搭建博客并深度优化

一、简介

hexo是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

二、环境配置及部署到github的教程（B站）

★★★传送门1 ：此教程只需看前面3P，讲的很好，但他使用的是yilla主题。
★★★传送门2 ：如果前面环境安装出问题可以看此教程P2，此教程使用的matery主题即为本博客使用的主题。
——看完以上两个教程，恭喜你已经建站成功了，并下载了matery主题(下载方式为：在博客文件目录下鼠标右击后点击“git bash here”，并输入以下指令)

git clone https://github.com/blinkfox/hexo-theme-matery.git

接下来是博客个性化设置1，个性化设置2以及（修改网站域名）腾讯云域名注册，默认域名是：你的github用户名.github.io

三、注意事项

环境配置需要安装git和nodejs
①nodejs下载教程：最后的测试可以略过，一般来说都能成功。
②git安装教程：切换页面是如果加载失败，需要手动点击reload
③你需要注册一个GitHub账号：github官网
在页面左上角有个绿色的“new”，点击会出现以下界面：
1处为仓库名称，一定要写和github用户名一样的名字，如图就是填：dxymygit.github.io
2、3、4默认就行，最后点击5

四、常见问题（如果未解决可至csdn官网查询）

1、插入看板娘教程
2、看板娘模型预览
3、markdown语法
4、中文包路径：themes\hexo-theme-matery\languages\zh-CN.yml
5、（由于部署博客的教程都为2019年的）此处的branch后应该由master改成main,否则会出现github上传后网站不更新的报错！
6、听歌板块部署教程 （部署音乐有吸底和首页显示可以自由选择）
7、修改域名
8、★★★域名映射教程

10、网站需要通过ssl认证（通过后会在www前生成https://），并备案（需要域名注册后三个工作日才可备案）,也是在腾讯云里操作
11、备案后可以进行cdn加速 （cdn加速教程） ,（★腾讯云cdn加速教程）
12、百度资源搜索平台
13、windows将nodejs版本降级
13、颜色代码表

五、核心调试代码（git bash here于博客根目录文件夹下）

hexo n “我的第一篇文章” 等价于 hexo new “我的第一篇文章” 还等价于 hexo new post “我的第一篇文章”

hexo p 等价于 hexo publish

hexo g 等价于 hexo generate

hexo s等价于 hexo server

hexo d 等价于 hexo deploy

hexo g -d等价于hexo generate –deploy

注: hexo clean 没有 简写, git –version 没有简写

六、常用快捷键

ctrl+s：保存文件
ctrl+a：全选
ctrl+f：查找
alt+tab：快速切换窗口
ctrl+alt+.:呼出任务管理器
vscode对于markdown文件的侧边预览：ctrl+shitf+v

七、文章front-matter介绍

Front-matter 选项详解
Front-matter 选项中的所有内容均为非必填的。但我仍然建议至少填写 title 和 date 的值。

配置选项 默认值 描述
title Markdown的文件标题 文章标题，强烈建议填写此选项
date 文件创建时的日期时间 发布时间，强烈建议填写此选项，且最好保证全局唯一
author 根 _config.yml 中的 author 文章作者
img featureImages 中的某个值 文章特征图，推荐使用图床(腾讯云、七牛云、又拍云等)来做图片的路径.如: http://xxx.com/xxx.jpg
top true 推荐文章（文章是否置顶），如果 top 值为 true，则会作为首页推荐文章
cover false v1.0.2版本新增，表示该文章是否需要加入到首页轮播封面中
coverImg 无 v1.0.2版本新增，表示该文章在首页轮播封面需要显示的图片路径，如果没有，则默认使用文章的特色图片
password 无 文章阅读密码，如果要对文章设置阅读验证密码的话，就可以设置 password 的值，该值必须是用 SHA256 加密后的密码，防止被他人识破。前提是在主题的 config.yml 中激活了 verifyPassword 选项
toc true 是否开启 TOC，可以针对某篇文章单独关闭 TOC 的功能。前提是在主题的 config.yml 中激活了 toc 选项
mathjax false 是否开启数学公式支持 ，本文章是否开启 mathjax，且需要在主题的 _config.yml 文件中也需要开启才行
summary 无 文章摘要，自定义的文章摘要内容，如果这个属性有值，文章卡片摘要就显示这段文字，否则程序会自动截取文章的部分内容作为摘要
categories 无 文章分类，本主题的分类表示宏观上大的分类，只建议一篇文章一个分类
tags 无 文章标签，一篇文章可以多个标签
